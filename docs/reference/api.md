# API Reference (Doxygen)

The full C API reference is generated by [Doxygen](https://www.doxygen.nl/) and published alongside this documentation.

[:material-book-open-variant: Open API Reference](../api/index.html){ .md-button .md-button--primary }

The API reference covers all public headers in `include/mrtklib/`:

| Header | Description |
|--------|-------------|
| `mrtklib.h` | Main include — pulls in all public headers |
| `mrtk_const.h` | Constants, system bitmasks, limits |
| `mrtk_types.h` | Core data types (`obsd_t`, `nav_t`, `prcopt_t`, etc.) |
| `mrtk_func.h` | Function declarations |
| `mrtk_toml.h` | TOML configuration loader |
| `mrtk_cssr.h` | Compact SSR (CSSR) decoder |
| `mrtk_madoca.h` | MADOCA-PPP L6E/L6D decoder |

!!! note "Build locally"
    To generate the API reference locally:

    ```bash
    sudo apt-get install doxygen graphviz  # or: brew install doxygen graphviz
    cmake --build build --target doc
    open build/doc/api/index.html
    ```
